# msxbase

msx base system rom

## Base

Baseline MSX system sources.

The sources can be assembled with z88dk's z80asm into a MSX 1 and MSX 2 BIOS/BASIC system rom.

## RCBUS

Customized MSX system sources for retrocomputers with RomWBW HBIOS.

Launch MSX BASIC and HBDOS (MSX-DOS 1 compatible) from CP/M.
